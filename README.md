# Lavida

Lavida is a lightweight, frameless YouTube link manager and organizer built with Python and PyQt6, designed specifically for Linux.

## Requirements

- Python 3.x
- PyQt6
- requests (for title fetching)

## Installation

1. Clone the repository or download the source code.
2. Install the required dependencies:
   pip install PyQt6 requests
3. Run the application:
   python main.py
